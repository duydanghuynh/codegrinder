Getting started on this assignment

You are now ready to access this problem from the grind tool. If you already have it set up, go to your project directory and use the following command to download this problem set:

grind get

Downloading the grind tool

Follow these instructions if you are installing grind for the first time, or if you are upgrading it to a new version. Click on the name of your operating system to open the instructions:

Linux

Type the following commands to download the grind tool. These instructions apply to downloading it for the first time, and also for upgrading it to a new version:

curl --compressed --create-dirs -o ~/bin/grind \
    /grind.linux
chmod 755 ~/bin/grind

Type grind and hit enter to make sure the command is recognized. If not (if you see the message "grind: command not found"), try logging out and back in again.

macOS

Type the following commands to download the grind tool. These instructions apply to downloading it for the first time, and also for upgrading it to a new version:

curl --compressed --create-dirs -o ~/bin/grind \
    /grind.macos
chmod 755 ~/bin/grind

Type grind and hit enter to make sure the command is recognized. If it does not work (if you see the message "grind: command not found"), run the following command:

echo 'export PATH=$PATH:$HOME/bin' >> ~/.bash_profile

Then log out and log in again. Type grind and hit enter to make sure the command is recognized.

Windows

Note: if you are using Windows 10, install Bash for Windows and then use the Linux instructions from a Bash shell. This is simpler and works better than the normal Windows method.

Follow these steps to set up CodeGrinder. To upgrade grind, you only need to repeat the 3rd step.

  1. Make a folder on your U: drive called winbin (or pick your own location and remember it)
  2. Add this directory to your Path using the Control Panel. Search online for help if you do not know how to do this
  3. Download the grind.exe tool by right-clicking here and selecting “Save link as” or “Save target as” to save it to that directory.

Logging in

After you install grind you must log in. You will normally only need to do this once per semester. To do so, type this command:

grind login

Note that the command given above is custom to you, and will expire 5 minutes from when you first loaded this page. If you need a fresh login token, reload this page.
